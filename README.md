# Sales Performance Dashboard (Excel) 

 This project showcases a fully interactive Sales Performance Dashboard built in Microsoft Excel using
 data cleaning, transformation, and visualization techniques widely used in Data Analytics.
## Project Overview
 The goal of this dashboard is to analyze sales performance across states, categories, and months to
 help identify high- and low-performing regions.

- What I Did- Cleaned and transformed raw sales data using Power Query- 
- Built a structured data model for accurate reporting- 
* Created KPIs for 
* Total Sales, 
* Total Orders
* Customer Count
 
 Designed interactive visuals including:- 
* State-wise sales map- 
* Category-wise insights- 
* Monthly trend analysis- 
* Built using PivotTables and PivotCharts for dynamic filtering
 
 ## Skills Used- 
 Microsoft Excel- Power Query- PivotTables & PivotCharts- Data Cleaning- Data Visualization- Dashboard Design
